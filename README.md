## Project 

This repo contains an UI clone (partial) from Discord application, based on Rocketseat's tutorial

### Techs

 - React.js
 - Styled Components
 - TypeScript

### Usage

- Run npm install or yarn install.
- Run yarn start and access http://localhost:3000.
